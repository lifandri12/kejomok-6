Tiga Metode CSS

1. Inline CSS

CSS yang ditulis langsung pada atribut style elemen HTML. Digunakan untuk styling khusus pada elemen tertentu. Contoh: <div style="color: blue; font-size: 16px;">

2. Tag CSS (<style>)

CSS yang ditulis dalam tag <style> di dalam <head>. Baik untuk styling halaman tertentu atau komponnya yang terbatas pada file HTML satu ini saja.

3. External CSS (File Terpisah)

CSS yang ditulis di file .css terpisah dan dimuat dengan tag <link>. Metode terbaik untuk reusability, maintainability, dan performance. File ini menggunakan metode ini!

Pembelajaran CSS

Inline CSS

Ditulis langsung di atribut style. Kelebihan: Cepat & spesifik. Kekurangan: Sulit di-maintain, tidak reusable.

Tag CSS

Dalam tag <style> di head. Kelebihan: Lebih rapi, scoped. Kekurangan: Tidak bisa di-share antar file.

External CSS

File .css terpisah. Kelebihan: Reusable, performa, mudah maintain. Kekurangan: Request tambahan (minimal dengan modern caching).
